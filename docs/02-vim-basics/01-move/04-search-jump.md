# Search & Jump

## Search forward
- In normal mode type /word_to_search
- Then press enter to search forward

## Search backward
- To search backward use ? instead of /

## Search with last search word 
- Simply / and press enter

## Search for word under current cursor
- Press *  

## Next occurence of prev occurence
- After pressing enter press n to go to next occurence and N to to go prev occurence

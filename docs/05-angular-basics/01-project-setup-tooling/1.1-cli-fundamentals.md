# 1.1 Angular CLI Fundamentals

**Branch: `1-1-angular-cli-fundamentals`**

Angular CLI is the official command-line interface for Angular development. It automates project setup, generates boilerplate code, builds applications, runs tests, and handles deployments.

## Core CLI Commands

### Project Generation

```bash
# Create new standalone app (Angular 17+ recommended)
ng new my-app --standalone --routing --style=scss --package-manager=npm

# Skip git init and install prompts
ng new my-app --skip-git --skip-install

# Generate application in existing workspace
ng generate application dashboard --routing --style=scss
```

### Component Generation

```bash
# Basic standalone component
ng generate component hero --standalone --inline-style --inline-template

# Full-featured component
ng g c heroes --export --lint-fix --dry-run

# Skip tests for quick generation
ng g c user/profile --skip-tests
```

### Other Schematics

```bash
# Services (tree-shakable)
ng g s hero --provided-in=root

# Directives
ng g directive highlight

# Pipes
ng g pipe duration

# Modules (legacy)
ng g module shared --routing
```

## Development Server

```bash
# Start dev server with live reload
ng serve --port 4201 --open

# Production-like build during development
ng serve --configuration production

# Proxy API calls to backend
ng serve --proxy-config proxy.conf.json

# SSL for HTTPS development
ng serve --ssl
```

## Build & Optimization

```bash
# Development build
ng build

# Production build (AOT + minification)
ng build --configuration production

# Base href for subdirectories
ng build --base-href /my-app/

# Watch mode for libraries
ng build my-lib --watch
```

## Testing & Quality

```bash
# Unit tests
ng test --watch --browsers=ChromeHeadless

# Coverage report
ng test --code-coverage

# E2E tests
ng e2e --configuration=ci
```

## Project Configuration

**angular.json** key sections:

```json
{
  "projects": {
    "my-app": {
      "architect": {
        "build": {
          "builder": "@angular-devkit/build-angular:browser",
          "options": {
            "outputPath": "dist/my-app",
            "index": "src/index.html"
          }
        }
      }
    }
  }
}
```

## Workspace Commands

```bash
# List all projects
ng build --help

# Generate library
ng generate library ui-components

# Build all applications
ng build --all

# Lint entire workspace
ng lint --all

# Update Angular globally
ng update @angular/cli @angular/core
```

## Essential Flags

| Flag           | Purpose                              |
| -------------- | ------------------------------------ |
| `--dry-run`    | Preview changes without generating   |
| `--lint-fix`   | Auto-fix linting issues              |
| `--skip-tests` | Skip test file generation            |
| `--export`     | Export component/service from index  |
| `--standalone` | Generate as standalone (no NgModule) |

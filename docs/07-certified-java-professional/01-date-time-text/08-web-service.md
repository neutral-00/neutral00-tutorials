# Web Service

Java is mainly used to build the backend systems of enterprise applications.


## SOAP Servies
SOAP services work with any transport. \
They carry data in xml format. \
To build soap services, Java provides JAXB and JAXP APIs.
    - you will get apis for XML object mappings, marshalling and unmarshelling.
    - JAX-WS API enables SOAP Service implementation.

## REST Services
Rest Services work with HTTP transport. \
They carry data in text/json/xml or any other format. \
To build rest services which works with json and xml, Java provides JSONP apis.
    - Using it you can do JSON object mappings, marshalling and unmarshalling.
    - JAX-RS API enables REST Service implementation.

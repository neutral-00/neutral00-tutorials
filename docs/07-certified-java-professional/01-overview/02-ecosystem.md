# Ecosystem

Widely used in 
- smartcards
- supercomputers
- IOT



There are different editions of Java for different ecosystems.
1. Java Card - for smart cards and sim cards
2. Java Micro Edition - for smart tvs and setop boxes
3. Java Standard Edition - for general purpose applications
4. Java Micro Profile - for Micro Services
5. Java Enterprise Edition - for enterprise java applications
    - mostly in the form of web servvices
    - enterprise java beans for sharing code

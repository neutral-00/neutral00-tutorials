# Thread methods

1. `start` method is used to start a thread.
2. `run` is the method where you override and write the actual code.
3. `sleep` is used to suspend the execution of the thread for a specified period.
4. `join` makes the calling thread to wait for the thread to complete.
5. `setPriority` sets the priority to the thread.
6. `interrrupt` method is use to interupt the thread.
7. `yield` method is used to give hint to the scheduler that cpu time may be allocated to others.
8. `setDaemon` is set to true to tell the schedule to terminate this thread if other user thread are completed.


**NOTE**
- Thread.MAX_PRIORITY is 10
- Thread.MIN is 1
- Thread.NORM is 5

# Terms and Definitions

## Data Structure

- Data structure is a storage unit for data.
- It uses specialized formats to manage data efficiently
    - By managing data I mean store, retrieve, sort, filter, update data.

## Algorithms

- Algorithms are step-by-step procedures for performing a task or
- formulas for solving a problem.
- It often operates on data organized by data structures.

Combining Data Structures and Algorithms help developers solve a lot of complex problems.

## Optimal Solution

A solution is considered by optimal by checking two factors:

1. Space Complexity 
1. Time Complexity

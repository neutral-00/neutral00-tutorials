## About Repository
Repository is a java interface that entends `JpaRepository`. \
Doing so will enable us to use common methods to save/retrieve/udpate/delete the entities.
Repository layer manages storage and access to the data layer.


## Steps
1. under `src\java\com\lousing` create a package `repository`
2. create the below interfaces

<TabItem value="user" label="UserRepository.java">

package com.lousing.repository;

import com.lousing.entity.User;
import org.springframework.data.jpa.repository.JpaRepository;

}

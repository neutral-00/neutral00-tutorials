# PostgreSql Setup

We are going to use Docker Desktop to setup postgres.

## Pre-requisites
1. Install Docker Desktop


## Steps
1. start Docker Desktop
2. open terminal and run the command below

```sh
docker run --name docker-postgres -p 5432:5432 -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=postgres -e POSTGRES_DB=postres -d postgres
```

# Define Interface

Define how the structure of models using interfaces.

## ng g interface

create folders as `src/app/features/models`
cd into it and run the below command to generate the interfaces

```sh
ng g interface User model
ng g interface Email model
ng g interface Fruit model
```

## Define The Models

Define the models as shown below:

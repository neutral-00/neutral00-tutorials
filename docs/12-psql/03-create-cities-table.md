# cities table

Open `https://pg-sql.com/` and run the below qeury to create the cities table.

```sql
CREATE TABLE cities (
	name VARCHAR(80),
  country VARCHAR(80),
  population INTEGER,
  area INTEGER
);
```

## Observation
Give attention that 
- Keywords are all in UPPERCASE
- Identifiers are all in lowercase

## Info
- INTEGER can store numbers ranging between negative 2 billion and positive 2 billion

### **Core Java Concepts for Interviews**

#### **1. Object-Oriented Programming (OOP)**
- **Core Principles**: Encapsulation, Inheritance, Polymorphism, Abstraction
- **Access Modifiers**: `public`, `private`, `protected`, `default`
- **Class vs Object**: Definitions and differences
- **Static vs Instance Members**
- **Overloading vs Overriding**
- **Abstract Classes vs Interfaces**
- **Inner Classes**: Static, Non-static, Anonymous

---

#### **2. Java Basics**
- **Data Types**: Primitive and Non-Primitive
- **Control Structures**: `if`, `else`, `switch`, loops (`for`, `while`, `do-while`)
- **Operators**: Arithmetic, Logical, Bitwise, Relational
- **Type Casting**: Implicit and Explicit
- **String Handling**: `String`, `StringBuilder`, `StringBuffer`

---

#### **3. Exception Handling**
- **Types of Exceptions**: Checked vs Unchecked
- **Keywords**: `try`, `catch`, `finally`, `throw`, `throws`
- **Custom Exceptions**
- **Best Practices**: Exception hierarchy, proper handling

---

#### **4. Collections Framework**
- **Core Interfaces**: `List`, `Set`, `Map`, `Queue`
- **Implementations**: `ArrayList`, `LinkedList`, `HashSet`, `TreeSet`, `HashMap`, `TreeMap`, `PriorityQueue`
- **Iterators**: `Iterator`, `ListIterator`, `for-each`
- **Comparators**: `Comparable` vs `Comparator`

---

#### **5. Multithreading and Concurrency**
- **Thread Lifecycle**: `new`, `runnable`, `running`, `waiting`, `terminated`
- **Thread Creation**: Extending `Thread` vs Implementing `Runnable`
- **Synchronization**: `synchronized` keyword, Locks
- **Concurrency Utilities**: `ExecutorService`, `Callable`, `Future`
- **Deadlocks and Race Conditions**

---

#### **6. Java 8 Features**
- **Functional Interfaces**: `Predicate`, `Consumer`, `Supplier`, `Function`
- **Lambda Expressions**
- **Streams API**: Filtering, Mapping, Reducing
- **Optional Class**
- **Default and Static Methods in Interfaces**

---

#### **7. Memory Management**
- **JVM Architecture**: Class Loader, Method Area, Heap, Stack, PC Register, Native Method Stack
- **Garbage Collection**: How it works, `finalize()` method
- **Memory Leaks**: Causes and prevention

---

#### **8. File I/O and Serialization**
- **File Handling**: `File`, `FileReader`, `FileWriter`, `BufferedReader`, `BufferedWriter`
- **Serialization and Deserialization**: `Serializable` interface, `transient` keyword
- **NIO Package**: `Path`, `Files`, `Channels`, `Buffers`

---

#### **9. Design Patterns**
- **Singleton**
- **Factory**
- **Builder**
- **Observer**
- **Dependency Injection** (basic understanding)

---

#### **10. Miscellaneous**
- **Enums**
- **Annotations**
- **Reflection API**
- **JDBC Basics**: Connecting to a database, executing queries
- **Unit Testing**: Basics of JUnit and Mockito

---

### **Suggested Study Plan**
1. Start with **Java Basics** and **OOP** concepts to build a strong foundation.
2. Move to **Exception Handling** and **Collections Framework** as they are frequently asked in interviews.
3. Study **Multithreading** and **Concurrency** for advanced-level questions.
4. Focus on **Java 8 Features** as they are highly relevant in modern development.
5. Learn **Memory Management** and **File I/O** for practical coding scenarios.
6. Understand **Design Patterns** and their use cases.
7. Practice coding problems and mock interviews to solidify your understanding.

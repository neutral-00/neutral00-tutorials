## Canvas Setup
- `width="160" height="120"`: Fixed display size
- `viewBox="0 0 160 120"`: Scalable coordinate system (0,0 top-left)
- Center point: (80,65)[1]

## Outer Cloud Shape (First Path)
```
M 80 25                    // Start at center-top (80,25)
C 105 25, 125 45, 125 65   // Right bulge (125,65)
C 125 90, 105 105, 80 105  // Bottom-right curve back to center
C 55 105, 35 90, 35 65     // Left bulge (35,65)  
C 35 45, 55 25, 80 25      // Close top curve to start
```
Creates puffy cloud outline spanning 90px wide √ó 80px tall.

## Inner Highlight (Second Path)
```
M 80 40                    // Inner highlight center
C 65 40, 50 52, 50 65      // Left inner curve
C 50 80, 65 90, 80 90      // Bottom inner curve  
C 95 90, 110 80, 110 65    // Right inner curve
C 110 52, 95 40, 80 40     // Close inner highlight
Z                          // Fill inner circle
```

## Key Features
- **`fill-rule="evenodd"`**: Outer path fills entire cloud, inner path **cuts out** highlight creating shine effect

# Angular Material Skill Checklist

Here are list of must have skillsets a senior Angular Material Developer must have.
All these skills will be tested in a git repo: [https://github.com/neutral-00/poc-angular-material](https://github.com/neutral-00/poc-angular-material)
They will be group by branch name. For example, skills under `Setup` will be tested under the git branch `setup`.

### Installation & Theme

# Angular Project Setup

Now let's setup angular project - the client facing application.

## Pre-requisites
install the angular cli: `pnpm add @angular/cli --global`

### Create Angular Project
```sh
ng new lousing-frontend
```

You can create the frontend project anywhere. \
It is indepependent.

Answer a few questions and your project should be created. \
cd into it and run `ng serve` to start the local dev server. \
navigate to http://localhost:4200

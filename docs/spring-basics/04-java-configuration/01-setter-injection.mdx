# Setter Injection

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

Modify the class implementing the service to have a setter method for the repository. \
Create `AppConfig.java` and call the setter method.

<Tabs>
<TabItem value="speakerServiceImpl" label="SpeakerServiceImpl.java" default>
```java title="src/main/java/org/lousing/service/SpeakerServiceImpl.java"
package org.lousing.service;

import org.lousing.model.Speaker;
import org.lousing.repository.SpeakerRepository;

import java.util.List;

public class SpeakerServiceImpl implements SpeakerService {
    // highlight-next-line
    private SpeakerRepository repository; // 1️⃣

    // highlight-start
    // 2️⃣
    public void setRepository(SpeakerRepository repository) {
        this.repository = repository;
    }
    // highlight-end
    
    @Override
    public List<Speaker> findAll() {
        return repository.findAll();
    }
}
```

:::note

1️⃣ Removed the repository instantiation \
2️⃣ Added setter for the repository

:::

<TabItem value="appConfig" label="AppConfig.java">

```java title="src/main/java/org/lousing/AppConfig.java" showLineNumbers
package org.lousing;

import org.lousing.repository.SpeakerRepository;
import org.lousing.repository.StubSpeakerRepositoryImpl;
import org.lousing.service.SpeakerService;
import org.lousing.service.SpeakerServiceImpl;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class AppConfig {

    @Bean(name="speakerRepository")
    public SpeakerRepository getSpeakerRepository() {
        return new StubSpeakerRepositoryImpl();
    }

    @Bean(name = "speakerService")
    public SpeakerService getSpeakerService() {
        SpeakerServiceImpl service = new SpeakerServiceImpl();
        service.setRepository(getSpeakerRepository());
        return service;
    }
}

```

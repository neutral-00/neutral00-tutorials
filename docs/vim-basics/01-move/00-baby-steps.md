# Baby Steps
Let's start with the baby steps. \
Navigate to any directory containing some text files. \
Open it with the command vim {name-of-file.txt}

Now, by default vim opens a file in normal mode.
- Use hjkl to move around
- Press i to start editing (insert mode) or I to insert at the beginning of the line.
- Press a to start editing (a is for append) Or A to append at the end of the line.
- Be back to normal mode by press ESC or ctrl + c
